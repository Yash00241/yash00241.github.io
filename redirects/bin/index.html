What Does Bin Do - Simplified

The Three Major Steps of Working of BIN

Select Parts

From the component selection menu, you can select desired components or do a roll faucet to get random items and try to create something through them.

Creation

You'll be redirected to Wokwi.com so that you can create connections and write code for the porject, and test it out virtually.

Ship em'

After submission of BIN project, your project items will be shipped to you and can be created on your desk.

About My BIN Project

This project is a bicycle speedometer built using a Raspberry Pi Pico, an MPU6050 accelerometer, an SSD1306 OLED display, and a Neopixel LED. The speedometer measures the bike's speed by calculating acceleration data from the MPU6050 and converts it into speed in kilometers per hour (km/h). The OLED display dynamically updates to show the current speed, ensuring real-time feedback for the rider.
